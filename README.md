# 🧠 Dysgraphia Detection using CNN

## 📌 Overview
This project aims to detect dysgraphia (writing disorder) using a deep learning model based on Convolutional Neural Networks (CNN).

## 🎯 Objective
To build an AI model that can classify handwriting images into normal or dysgraphia cases.

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- NumPy

## 📊 Results
- Accuracy: 98.34%
- Model: ResNet50 + CNN head

## 📂 Dataset
Dataset includes handwritten characters categorized into:
- Normal
- Reversal

## 🚀 How to Run
1. Clone the repo  
2. Install requirements  
3. Run the notebook
